Reinforcement Learning: Autonomous car

Deep Q-Network for Autonomous car

Description

This page loads a trained Deep Q-Network (DQN) and uses it to drive a car along a predefined track. The training is done in Node.js using tfjs-node.

Algorithm

A DQN is trained to estimate the value of actions given the current game state. The DQN is a 1D convolutional network. See dqn.js.
